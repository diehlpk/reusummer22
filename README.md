Scripts to comapre the simulations including damage. 

## Dependencies

* [numpy](https://numpy.org/)
* [matplotlib](https://matplotlib.org/)

## Usage

```bash
python3 -m venv deps
source deps/bin/activate
pip install -r requirements.txt
```
